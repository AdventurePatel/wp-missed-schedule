# wp-missed-schedule
Find only missed schedule posts with this issue, every 15 minutes, and republish correctly 10 items each session. Plugin no longer available on WordPress.org for explicit author request. Work perfectly on clean WordPress 2.1+ to 4.7+ ~ 4.8-RC1 (90.000+ installs 300.000+ downloads on 2016-04-13) Warning: not install unauthorized forked copy! !!
